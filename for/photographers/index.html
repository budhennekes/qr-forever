Why your gallery code should never expire

Most QR generators make dynamic codes: the code points at their server, which forwards clients to your gallery. That works until the free trial ends or the subscription lapses. Your cards and welcome packets outlive any subscription, and a dynamic code dies with the plan and takes the gallery link with it.

A static code bakes the URL into the pattern itself, permanently. Nobody can turn it off, including me. The full story is in do QR codes expire, and the card-specific details are in the business card guide.

Questions photographers ask

What size should the QR code be on a business card?

0.6 to 0.8 inch works for an in-hand scan, as long as the quiet margin around the code stays intact. The business card guide covers placement and sizing in detail.

My gallery links change with every client. How do I handle that?

Two ways. Make a fresh code per shoot, which takes seconds and costs nothing. Or point one code at a client-login page you control, so the same code on every card always lands somewhere useful.

Can I put my logo in the middle?

Yes. Add a logo in the center and the tool automatically raises the code's error correction so it still scans. Details in the logo size guide.

Will it match my brand?

Custom colors, four dot styles, and a transparent background for dark card stock. If your color combination gets hard to scan, you get a contrast warning before you download.

Do I need an account or a subscription?

No. The tool runs in your browser, the code is free, and there is nothing to renew. Your links and logo never leave your device.
